# Fram_MVP

Testnettside for FRAM NTNU.

## Innhold

- `index.html` – Hoveddokumentet for nettsiden. Åpne denne filen i en nettleser for å se testsiden.

## Kom i gang

1. Last ned eller klon dette repositoriet.
2. Åpne `index.html` i en nettleser.

## Om prosjektet

Dette er en MVP (Minimum Viable Product) for FRAM sin nettside ved NTNU. Siden er bygget med enkel HTML og CSS uten eksterne avhengigheter.
